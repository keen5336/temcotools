MARS Label Tool

Paste the full return page text, extract the fields automatically, preview the label, and print straight to a networked label printer. Includes editable ZPL templates stored in local storage.

Ready

1) Paste source text

From the return page: Ctrl + A, Ctrl + C, paste it here, then parse. The parser grabs the next non-empty line after each field label.

2) Extracted fields

These are the values that feed the label. You can correct anything manually before printing.

3) Printer

Use direct raw printing only if your browser and printer allow it. Otherwise point this at a tiny local print bridge like http://localhost:3000.
